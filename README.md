#  NimmaGuru - Rural Guidance Android App

NimmaGuru is an Android application designed to connect villagers with local experts (“Gurus”) who can provide guidance, support, and useful information in different fields such as farming, education, healthcare, government schemes, and daily life assistance.

The application focuses on creating a simple, user-friendly, and accessible platform for rural communities.

---

## Features

-  Guru Profiles
-  Different Village Support
-  Search and Filter Gurus
-  Contact Information
-  Modern Android UI
-  Detailed Guru Information
-  Fast and Responsive Design
-  Easy Navigation

##  Getting Started

### 1️ Clone the Repository

```bash
git clone https://github.com/poornima-badiger/NimmaGuru.git
```

---

### 2️ Open in Android Studio

- Open Android Studio
- Click **Open**
- Select the `NimmaGuru` project folder

---

### 3️ Sync Gradle

Wait for Android Studio to complete Gradle sync.

---

### 4️ Run the Application

- Connect Android device OR start emulator
- Click ▶️ Run button
